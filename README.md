# Device Collection

Веб-приложение для управления коллекцией устройств. Добавляй карточки девайсов, фильтруй по категориям, сортируй список и находи нужные устройства через поиск.

Github pages: https://nomercyzzz.github.io/Device-Collection-Vue/

## Возможности

- ✨ **Добавление карточек** — создавай карточки с названием, категорией, описанием и фото
- 🔍 **Поиск** — быстро находи устройства по названию
- 🏷️ **Фильтрация** — фильтруй девайсы по категориям
- 🖼️ **Загрузка изображений** — добавляй фото устройств (PNG, JPEG, WebP)
- 🎨 **Современный UI** — минималистичный чёрный интерфейс на основе Tailwind CSS
- ⚡ **Плавная анимация** — переходы и эффекты для улучшенного UX

## Технологический стек

- **Vue 3** — фреймворк
- **Vite** — сборщик и dev-сервер
- **Tailwind CSS** — стилизация
- **gh-pages** — деплой на GitHub Pages

## Установка и использование

```bash
# Установка зависимостей
npm install

# Локальная разработка
npm run dev

# Сборка для production
npm run build

# Деплой на GitHub Pages
npm run deploy
